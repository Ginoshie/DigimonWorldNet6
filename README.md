## :arrow_right: Installation
1. Choose the version you want to download from the [release page](https://github.com/Ginoshie/DigimonWorldNet6/releases)
2. Download the file named 'DW1Tool-win-Setup.exe'
3. Open the .exe and 
4. A "Windows protected your PC" warning popups, click 'Run anyway'
   <br><img width="400" height="400" alt="image" src="https://github.com/user-attachments/assets/e32d2c8f-5730-4080-a107-a1715e0444af" />
5. The installer screen shows up and installs the tool in `%LocalAppData%\DW1Tool`
6. The app automatically starts
7. Enjoy the app! 🥳


<br/>
