## :arrow_right: First time usage
1. Download the version you want to use from the [release page](https://github.com/Ginoshie/DigimonWorldNet6/releases)
2. Run the .exe
3. A "Windows protected your PC" warning popups, click 'More info'
<img width="400" alt="Windows protection popup" src="https://github.com/user-attachments/assets/595207a4-2e88-4a82-a3b4-9e0873cea9d9" />

4. Click Run anyway
<img width="400" alt="Run anway button" src="https://github.com/user-attachments/assets/3cfc05bd-f742-444a-8a9b-60f412fa8d12" />

5. Enjoy the app! 🥳
